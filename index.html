How to buy (agents & devs)

  1. POST a generation endpoint → you get 402 with payment requirements.
  2. Pay with any x402 client and retry — @x402/fetch does both automatically.
  3. Response returns { job_id, status_url } immediately (async).
  4. Poll GET /jobs/:id (free) until succeeded → download video_url within 24h.
import { wrapFetchWithPayment, x402Client } from "@x402/fetch";
import { ExactEvmScheme, toClientEvmSigner } from "@x402/evm";

const client = new x402Client().register("eip155:*", new ExactEvmScheme(signer));
const fetchPay = wrapFetchWithPayment(fetch, client);

const res = await fetchPay(GATEWAY + "/generate/seedance/5s-720p", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ prompt: "a corgi surfing a wave at sunset, cinematic" }),
});
const { job_id } = await res.json(); // then poll GATEWAY + "/jobs/" + job_id

Content policy

Prompts are screened before payment — rejected requests are never charged.
Hard red lines: content involving minors and real-person likeness / deepfakes. Sexually explicit content and graphic violence are rejected.
