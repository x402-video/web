How the live quote works

Every time you change a parameter, this page sends a real POST to https://api.x402video.com/generate/{model}/custom without a payment header. The server responds with HTTP 402 and a JSON body that includes the per-second rate for your chosen resolution. The price you see is computed from that live response — not a static table.

When you actually call the API with payment, the 402 quote will match exactly. You sign the EIP-3009 USDC authorization for precisely that amount — no overcharging, no rounding surprises.
